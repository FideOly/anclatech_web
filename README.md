## 🛠️ Cómo empezar

### Local (Desarrollo):
1. Clona o descarga este repositorio
2. Abre `index.html` con Live Server en VS Code
3. Los cambios se reflejan en tiempo real

### Producción:
1. Configura Formspree en contacto.html (línea con `action`)
2. Actualiza dominio en meta tags (todas las páginas)
3. Sube archivos a tu hosting/servidor
4. Configura HTTPS en tu proveedor
5. Actualiza DNS con CNAME.template

---

## 🔧 Configuración Importante

### **Formulario de Contacto:**
En `contacto.html`, actualiza:
```html
<form action="https://formspree.io/f/YOUR_FORM_ID">
```
Obtén tu Form ID en [formspree.io](https://formspree.io)

### **Dominio:**
En todas las páginas, actualiza:
```html
<meta property="og:url" content="https://tudominio.com">
```

### **Email de Contacto:**
En `aviso-de-privacidad.html`:
```html
<a href="mailto:tuemailreal@anclatech.com">contacto@anclatech.com</a>
```
